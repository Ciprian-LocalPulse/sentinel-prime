# Public Edition Notice

## Purpose

This repository is the Public Edition of Sentinel Prime.

It exists to provide a procurement-safe, public-facing view of the dashboard UI kit, brand, governance posture, and evaluation surface.

## Public Edition Includes

The Public Edition may include:

- static landing page
- simulated dashboard preview
- public visual assets
- non-sensitive product overview
- ownership and license notices
- support and security policies
- contribution rules
- public funding configuration

## Public Edition Excludes

The Public Edition does not include:

- real telemetry pipelines
- detection rules
- operational threat intelligence feeds
- client environments
- token generation methods
- private playbooks
- prompts or AI orchestration details
- API keys, secrets, credentials, or deployment variables
- customer, analyst, endpoint, or incident data
- commercial implementation details
- production security logic
- restricted private enterprise materials

## Private Enterprise Edition

The Private Enterprise Edition is separate from this public repository.

Private package access, private source review, customization support, deployment rights, commercial usage rights, white-label rights, marketplace distribution rights, exclusivity, and support obligations are available only through explicit written agreement.

The public repository should not be treated as the complete product, a complete source package, a security monitoring system, a threat detection system, a compliance reporting system, or a production deployment package.

## Public Discussion Boundary

Public issues and pull requests must remain limited to the public documentation and public preview surface.

Do not request or disclose telemetry pipelines, detection rules, client environments, token generation methods, private playbooks, prompts, credentials, customer data, endpoint data, incident data, internal strategy, or restricted implementation details in public GitHub channels.

## Procurement-Safe Interpretation

This repository may support initial review, but it is not:

- a SIEM
- an EDR or XDR product
- a SOAR platform
- an IDS or firewall
- a vulnerability scanner
- a managed security service
- a security audit
- a compliance certification
- a production readiness attestation
- a transfer agreement
- an open source release

Formal diligence, security review, legal review, implementation review, or enterprise access review must occur privately under appropriate written terms.
