# Support

## Public Support Scope

Public support for Sentinel Prime is limited to:

- public repository navigation
- public documentation questions
- public landing page issues
- broken public links
- clarification of the Public Edition boundary
- responsible disclosure routing

Public support does not include private implementation support, deployment guidance, production security integration, SIEM or EDR integration support, telemetry pipeline review, detection rule review, incident data review, or commercial implementation detail.

## Private Enterprise Discussions

Private enterprise access, licensing, acquisition, design-partner, customization, or implementation discussions must be handled outside public GitHub channels.

Schedule a private discussion:

https://cal.com/ciprian-stefan-plesca

## Security Support

For security concerns, do not open a public issue.

Follow [SECURITY.md](SECURITY.md).

## Response Expectations

This repository does not provide guaranteed public support response times.

Any paid support, private enterprise assistance, customization support, implementation support, marketplace support, or transfer support must be defined in a separate written agreement.

## Public Communication Rules

Do not post any of the following in public issues, discussions, or pull requests:

- secrets, tokens, keys, or credentials
- telemetry pipelines
- detection rules
- customer, analyst, endpoint, or incident data
- production security logic
- private playbooks
- prompts or AI orchestration details
- private enterprise materials
- third-party confidential information
